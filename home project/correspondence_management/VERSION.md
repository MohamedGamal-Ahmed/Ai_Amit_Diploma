## 🛠️ متطلبات النظام

### الحد الأدنى:
- **نظام التشغيل:** Windows 10, macOS 10.14, Linux Ubuntu 18.04
- **Python:** 3.8 أو أحدث
- **الذاكرة:** 512 MB RAM
- **التخزين:** 100 MB مساحة فارغة

### المستحسن:
- **نظام التشغيل:** Windows 11, macOS 12+, Linux Ubuntu 20.04+
- **Python:** 3.10 أو أحدث
- **الذاكرة:** 2 GB RAM
- **التخزين:** 500 MB مساحة فارغة
- **الشاشة:** 1366x768 أو أعلى
